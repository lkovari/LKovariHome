Angular v16 one of new feature is the Signals

There is a following example, simple arithmetic calculations with Signals

Here is the arithmetic operation, amount = payment * quantity - write-off.

When one of the values (payment, quantity, write-off) is changed then automatically calculate the amount, like as computed. Listening to the value changes of the form, "this.form.valueChanges.subscribe((value) =>" and set each value to it's own signal variable, for example "this.quantitySignal.set(value.quantity!);"

Only one form was built, so every value change will affects the amounts on both UI. components

The Code: 

    quantitySignal = signal(0);
    paymentSignal = signal(0);
    writeoffSignal = signal(0);
    amountSignal: Signal<number> = computed(() => 
        (this.paymentSignal() 
        * this.quantitySignal()) 
        - this.writeoffSignal());

On the left side you can see four numeric input fields with PrimeNG UI., and on the right side can see the same input fields with Material UI. components.
